README

Toolbar icons (in toolbar/ subdirectory) and the config icons (in config/ subdirectory) are from the Tango project (http://tango.freedesktop.org). They originally were released under the Creative Commons Attribution Share-Alike license (https://creativecommons.org/licenses/by-sa/2.5/) but since then have been released to the public domain. They were renamed to match the names of the gtk system icons they replace.

The toolbar icons "arrow-up-square.svg" and "eye-slash.svg" are from the Bootstrap icons (https://icons.getbootstrap.com/).
